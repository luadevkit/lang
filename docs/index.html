class([name][, super][, ...])

Creates a new class.

Creates a class with the given name, parent, and mixins. Mixins are included into the new class in the order they are passed to the function.

Parameters:

  • name string the name of the new class. (optional)
  • super class the parent of the new class. (optional)
  • ... mixin the mixins for the new class. (optional)

Returns:

    class the new class.

Raises:

if different mixins define values with the same name but different types.

mixin(name)

Creates a mixin with a given name.

Mixins are like classes but cannot be instantiated.

Parameters:

  • name string the name of the new mixin.

Returns:

    table a new mixin.
name_of(class_or_mixin)

Returns the name of a given class or mixin.

Parameters:

  • class_or_mixin class or mixin the type to get the name of.

Returns:

    string the name of the given type, or nil if the input argument is nil.

Raises:

if the given argument is not a class, a mixin, or nil.
